This is a Wireless Sensor network Application for Tracking a mobile entity in an indoor environment.
This Project was carried out at the Bits Pilani K K Birla Goa Campus.

this network consists of 5 types of motes
	1. Base Station
	
	2. Cluster Head
	
	3. Relay mote
	
	4. Sensor mote
	
	5. Mobile mote
	
	
The aim of the network is to track the position of the mobile mote in real time.

This is the code for a Cross Layered Protocol Stack and implements the following Protocols.

	A. Time Synchronization
	
	B. Periodic and Query based Routing
	
	C. Media Access Control
